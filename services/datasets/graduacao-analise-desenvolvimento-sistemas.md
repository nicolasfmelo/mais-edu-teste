# Graduacao em Analise e Desenvolvimento de Sistemas

## Informacoes gerais
- Nivel: Graduacao
- Modalidade: Remoto
- Tempo de formacao: 2,5 anos

## O que voce vai aprender
O curso aborda logica de programacao, desenvolvimento web, bancos de dados, APIs, testes, modelagem de sistemas e fundamentos de cloud. O aluno aprende a criar solucoes digitais, documentar requisitos, desenvolver software e participar de projetos com foco em qualidade e entrega continua.

## Onde aplicar no mercado de trabalho
Analise e Desenvolvimento de Sistemas prepara para atuar em empresas de tecnologia, consultorias, startups, equipes de produto e setores de transformacao digital. O curso te prepara para funcoes como desenvolvedor full stack junior, analista de sistemas, analista de suporte tecnico, analista de QA e integrador de sistemas.

## Grade curricular
- Logica de Programacao
- Algoritmos e Estruturas de Dados
- Programacao Orientada a Objetos
- Banco de Dados
- Desenvolvimento Front-end
- Desenvolvimento Back-end
- Engenharia de Requisitos
- Testes de Software
- APIs e Integracoes
- Cloud e DevOps
