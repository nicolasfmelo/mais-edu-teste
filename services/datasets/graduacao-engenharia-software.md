# Graduacao em Engenharia de Software

## Informacoes gerais
- Nivel: Graduacao
- Modalidade: EAD
- Tempo de formacao: 4 anos

## O que voce vai aprender
O curso aprofunda arquitetura de sistemas, modelagem, qualidade, testes, seguranca, engenharia de requisitos e gestao de desenvolvimento. O aluno aprende a projetar software escalavel, documentar solucoes, trabalhar com boas praticas de engenharia e colaborar em produtos digitais complexos.

## Onde aplicar no mercado de trabalho
Engenharia de Software prepara para atuar em empresas de tecnologia, fintechs, healthtechs, consultorias e squads de produto. O curso te prepara para funcoes em desenvolvimento de software, engenharia de qualidade, analise tecnica, arquitetura junior, engenharia de plataforma e suporte a ciclos de entrega.

## Grade curricular
- Fundamentos de Engenharia de Software
- Modelagem de Sistemas
- Arquitetura de Software
- Banco de Dados Avancado
- Desenvolvimento Web e Mobile
- Testes Automatizados
- Seguranca de Aplicacoes
- DevOps e Integracao Continua
- Gestao de Projetos de Software
- UX para Produtos Digitais
