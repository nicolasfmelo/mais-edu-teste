# Pos-graduacao em Ciencia de Dados

## Informacoes gerais
- Nivel: Pos-graduacao
- Modalidade: EAD
- Tempo de formacao: 12 meses

## O que voce vai aprender
O curso aprofunda estatistica aplicada, analise exploratoria, machine learning, visualizacao, engenharia de dados e interpretacao de modelos. O aluno aprende a transformar bases em insights acionaveis, criar pipelines analiticos e apoiar decisoes orientadas por dados.

## Onde aplicar no mercado de trabalho
Ciencia de Dados prepara para atuar em empresas de tecnologia, bancos, varejo, educacao, saude e consultorias analiticas. O curso te prepara para projetos de inteligencia de mercado, previsao de demanda, analise de comportamento, criacao de dashboards, modelos preditivos e suporte a areas de BI.

## Grade curricular
- Estatistica Aplicada
- Python para Analise de Dados
- SQL e Modelagem Analitica
- Visualizacao de Dados
- Machine Learning
- Engenharia de Dados
- ETL e Qualidade de Dados
- Modelos Preditivos
- Storytelling com Dados
- Projeto Aplicado em Analytics
